# 🚀 VIBETHON – AI/ML Interactive Learning Platform

## 📌 Overview

An interactive web-based platform designed to make **Artificial Intelligence and Machine Learning (AIML)** learning engaging, practical, and easy to understand.
The platform focuses on **hands-on experience**, allowing users to learn concepts, practice them, and explore real-world applications in a fun and interactive way.

---

## 🎯 Problem Statement

Design and develop a platform that:

* Simplifies complex AIML concepts
* Encourages interactive and experiential learning
* Provides hands-on practice and real-world simulations
* Enhances engagement using gamification elements

---

## 💡 Key Features

* 🔐 **User Authentication** – Secure login & user tracking
* 📚 **Structured Learning Modules** – Beginner to Advanced levels
* 💻 **Interactive Coding Playground** – Run and modify code in-browser
* 🎮 **Mini-Games** – Learn concepts through interactive games
* 📝 **Quiz System** – Instant feedback and scoring
* 🌍 **Real-World Simulations** – Practical AIML use cases
* 📊 **Progress Dashboard** – Track learning and performance
* 🏆 **Leaderboard & Gamification** – Badges, scores, and rankings
* 📱 **Responsive Design** – Works across all devices

---

## ⚙️ Tech Stack

* **Frontend:** Vite + React
* **Styling:** Tailwind CSS
